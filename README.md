# walmart-retail-analytics
Exploratory and financial analysis of Walmart retail sales with SQL, Python, and Business Intelligence.

## Overview
This project analyzes Walmart weekly sales data to identify the main drivers of revenue, seasonal patterns, and the impact of economic indicators.

The goal is to simulate a real-world data analysis project from raw data to actionable business insights.

## Dataset
The dataset contains weekly sales data for multiple Walmart stores including columns like:

- Store: Store number
- Date: Sales week start date
- Weekly_Sales: Sales
- Holiday_Flag: Mark on the presence or absence of a holiday
- Temperature: Air temperature in the region
- Fuel_Price: Fuel cost in the region
- CPI: Consumer price index
- Unemployment: Unemployment rate

Total records: 6 435

## Key Business Questions

- Do holidays significantly impact weekly sales?
- Which external factors affect sales?
- Are there strong seasonal patterns?
- Which stores generate the most revenue?
